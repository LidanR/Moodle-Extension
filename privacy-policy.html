מדיניות פרטיות - JCT Moodle Extension

עדכון אחרון: דצמבר 2024

1. איסוף מידע

התוסף אינו אוסף ואינו משתף שום מידע אישי עם שרתים חיצוניים. כל המידע נשמר מקומית בדפדפן שלך בלבד.

2. אחסון מידע

התוסף משתמש ב-chrome.storage לאחסון מקומי של:

  • העדפות צבעים מותאמות אישית
  • לוח זמנים שבועי של קורסים
  • מטמון נתוני מטלות (פג תוקף אחרי 5 דקות)
  • קורסים מועדפים
  • הגדרות מעקב מטלות

כל המידע נשמר רק בדפדפן שלך ולא מועבר לאף שרת חיצוני.

3. גישה לאתרים

התוסף מבצע בקשות רשת רק ל-moodle.jct.ac.il כדי:

  • לשפר את ממשק Moodle
  • לשלוף נתוני מטלות מה-API של Moodle
  • לבדוק סטטוס הגשות

כל הבקשות משתמשות באימות הקיים שלך ב-Moodle ולא נשלח מידע נוסף.

4. שיתוף מידע

התוסף אינו משתף שום מידע עם צדדים שלישיים. כל המידע נשאר מקומי בדפדפן שלך.

5. אבטחה

כל המידע נשמר מקומית בדפדפן שלך באמצעות Chrome Storage API. אין שרתים חיצוניים ואין העברת מידע.

6. שינויים במדיניות

אם נשנה מדיניות זו, נעדכן את התאריך בראש העמוד.

7. יצירת קשר

לשאלות או בקשות בנוגע לפרטיות, אנא צור קשר דרך Chrome Web Store.

Privacy Policy - JCT Moodle Extension

Last Updated: December 2024

1. Information Collection

This extension does not collect or share any personal information with external servers. All data is stored locally in your browser only.

2. Data Storage

The extension uses chrome.storage for local storage of:

  • Custom color preferences
  • Weekly course schedules
  • Cached assignment data (expires after 5 minutes)
  • Favorite courses
  • Assignment tracking settings

All data is stored only in your browser and is never transmitted to any external server.

3. Website Access

The extension makes network requests only to moodle.jct.ac.il to:

  • Enhance the Moodle interface
  • Fetch assignment data from Moodle's API
  • Check submission status

All requests use your existing Moodle authentication and no additional information is sent.

4. Information Sharing

The extension does not share any information with third parties. All data remains local in your browser.

5. Security

All data is stored locally in your browser using Chrome Storage API. There are no external servers and no data transmission.

6. Policy Changes

If we change this policy, we will update the date at the top of this page.

7. Contact

For questions or requests regarding privacy, please contact us through Chrome Web Store.
